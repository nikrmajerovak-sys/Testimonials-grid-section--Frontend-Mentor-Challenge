### Testimonial Grid Section

This is a solution to the Testimonial Grid Section challenge on Frontend Mentor. The goal of this project was to build a responsive testimonial layout using modern CSS techniques.

## 🚀 Overview
### 📌 The challenge

Users should be able to:

View the optimal layout depending on their device's screen size
See a responsive grid of testimonial cards
🛠️ Built with
HTML5
CSS3
CSS Grid
Flexbox
Mobile-first workflow
